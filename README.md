# Heart-Disease-prediction
# Description

This project focuses on predicting the likelihood of heart disease using machine learning techniques based on the UCI Heart Disease dataset. The model is built using ensemble learning methods to improve prediction accuracy and reliability.

# Features
🔹 Data preprocessing and feature selection
🔹 Ensemble models (Random Forest + Gradient Boosting)
🔹 High accuracy classification model
🔹 Model saved using Pickle (joblib)
🔹 Web API built with Flask
🔹 User input form for real-time prediction
🔹 Simple and interactive UI


# Tech Stack
Python
Pandas, NumPy
Scikit-learn
Flask
HTML, CSS


# Input Features
Age
Sex
Chest Pain Type (cp)
Resting Blood Pressure (trestbps)
Cholesterol (chol)
Fasting Blood Sugar (fbs)
Rest ECG (restecg)
Maximum Heart Rate (thalach)
Exercise Induced Angina (exang)
Oldpeak
Slope
CA
Thal
# Output
0 → No Heart Disease
1 → Heart Disease Detected
